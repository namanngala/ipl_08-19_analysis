Data Cleaning which is required¶

i.Pune was represented by various Team Names as 'Rising Pune Supergiant', 'Pune Warriors' & 'Rising Pune Supergiants' so as a convinience we will changes these with the recent team representing Pune 'Rising Pune Supergiant'

ii.Earlier the team name for delhi was 'Delhi Daredevils' but later it was changed to 'Delhi Capitals' so we will replace the "delhi Daredevils' with 'Delhi Capitals'

iii.Bangalore was Renamed as Bengaluru in 2014 so we will change the Name for City Bangalore to Bengaluru

In [125]:
